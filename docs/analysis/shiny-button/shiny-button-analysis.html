1. Standard Button (@property declarations)
Click Me
@property --gradient-angle registers a custom property as animatable angle type. Without it, CSS treats custom properties as strings, preventing smooth transitions.
2. Border-Spin Animation (conic-gradient rotation)
Rotating Border
@keyframes border-spin rotates --gradient-angle from 0deg to 360deg, creating a light that sweeps 360° around the button's border every 2.5 seconds.
3. Particle Shimmer (::before with radial gradient + mask)
With Sparkles
::before creates tiny white dots (0.5px) masked by conic-gradient to show only when the mask's visible wedge passes over them. Creates synchronized sparkle effect.
4. Directional Shimmer (::after with linear gradient + radial mask)
Diagonal Sweep
::after creates a diagonal gradient that translates across a 3x3 grid. Masked by radial-gradient (circle at bottom) to create a cone shape, simulating light from above.
5. Breathing Inner Glow (span::before with breathe animation)
Glowing Text
span::before creates a radial glow behind text that pulses opacity 0.3→0.8 over 4.5 seconds. Provides inner ambient light that breathes independently from border animation.
6. Mask-Image Control (visibility controller)
Masked Effects
mask-image != background. It's a visibility mask: black=visible, transparent=hidden. Conic gradient mask rotates with animation to spotlight effects. Radial mask creates cone shape for directional shimmer.
Variation: Different Offset (wider light strip)
Offset Gradient
--gradient-angle-offset: 30deg shifts the gradient starting point, widening the visible light portion of the conic-gradient.
Variation: Custom Shimmer Color (Cyan)
Cyan Shimmer
--gradient-shine: #06b6d4 changes the primary gradient color in conic-gradient from blue to cyan.
Variation: Slower Animation (4s vs 2.5s)
Slower Rotation
Animation duration changed to 4s makes the border-spin effect rotate more slowly, changing the perceived speed of the light sweep.
